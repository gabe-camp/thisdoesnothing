# thisdoesnothing
This is a python package (or will be) which does nothing but whose purpose is to provide a playground to practice implementing various python package development features.
Such as:
  * pyproj.toml 
  * setup.py
  * setup.cfg
  * .gitignore
  * python-semantic-release
  * pre-commit
  * tox
  * pytest
  * flake8
  * pydoc
